# AI-based Smart Road Crossing Assistant for Visually Impaired

This project is a Flask-based web application that uses YOLOv8 for real-time object detection to assist visually impaired individuals in crossing roads safely.

## Features
- Real-time video monitoring from webcam.
- Object detection for various entities (vehicles, people, animals, etc.).
- Admin panel for user management.
- Dynamic and modern UI.

## Technology Stack
- **Backend:** Python (Flask)
- **Frontend:** HTML, CSS, JavaScript
- **AI Model:** YOLOv8
- **Database:** SQLAlchemy

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Aryan210804/AI-Based-Smart-Road-Crossing-Assisted-For-Visual-Impaired-.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python app.py
   ```
